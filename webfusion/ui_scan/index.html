WebFusion Scanner

Shift-left web security checks for your CI pipeline.
⚠ Authorized targets only — scan sites you own or have written permission to test.
Hosted scan is passive-only (security headers, cookies, CORS, info leaks) and blocks internal/localhost targets. For active checks (XSS/SQLi/redirect/traversal) run the CLI: python -m webfusion.scan URL --active
